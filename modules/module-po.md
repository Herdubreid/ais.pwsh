## Celin.AIS.Po

Module for extracting App Procession Options.

### Get-Celin.AIS.Po

Get Procession Options.

_Application_
: The app name.

_Version_ (Optional)
: The version name.  Defaults to "ZJDE0001"

_WithApp_ (Optional)
: A flag that when set, adds the app name and version to the result.
